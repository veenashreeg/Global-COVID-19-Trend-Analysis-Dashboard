# 🌍 Global COVID-19 Trend Analysis Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard that analyzes global COVID-19 trends from 2020 to 2022.

## 🛠 Tools Used
- Power BI
- Excel
- DAX

## 📊 Dashboard Features
- Cases by Year
- Country Filter
- Date Filter
- Interactive World Map

## 📁 Files Included
- COVID19.pbix
- Dataset.xlsx
- Dashboard.png
